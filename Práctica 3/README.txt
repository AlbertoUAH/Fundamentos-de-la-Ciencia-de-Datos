Para visualizar el arbol del random forest:
1. Si se esta trabajando con RStudio, no es necesario ningun ajuste adicional
2. En caso de trabajar con el editor estandar de R, se debe incorporar la libreria reprtree.zip manualmente con el comando install.packages()
   debido a que el repositorio donde se encuentra ubicada dicha libreria pertenece al entorno de RStudio.
